## 4. Sumário Executivo: Racional Híbrido

Empresas adotam **Hybrid Identity Bridge** quando:

**Agilidade (Cloud RH):** OrangeHRM em PaaS oferece auto-scaling, HA nativa, patching automático – reduzindo de 4h para 15min o tempo de recovery de sistemas RH.

**Segurança (IGA Local):** midPoint permanece on-premises para controle granular de provisioning em sistemas legados (AD, Unix, mainframes) sem exposição direta à internet.

**Governança:** Shadow API atua como Identity Proxy com validação OAuth2 + Least Privilege, isolando tráfego sensível via Tailscale MeshVPN.

**Ganho Líquido:** 3x faster RH operations + Zero Trust Network Access entre Cloud/Local + Conformidade AZ-305 (Hybrid Connectivity, PaaS Security).
